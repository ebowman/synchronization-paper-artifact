# Papers

Preprints and manuscripts of the synchronization research series, with
their review status. **None of these documents has been peer reviewed.**
Each PDF here is (and will remain) the *submitted/working* version: per
publisher policy, public copies are never replaced with accepted or
typeset journal versions. If errors are found, corrections are issued as
new versions of this archive with an explicit correction note (see
[CORRECTION.md](../CORRECTION.md) for precedent).
